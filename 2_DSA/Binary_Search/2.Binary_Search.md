public static int sqrt(int N){

    // Edge case
    if (N == 0 || N == 1) {
        return N;
    }

    int i = 1;

    while(i * i <= N){
       i++;
    }

    return i - 1;

}

```

public static int sqrt(int N){
     if (N == 0 || N == 1) {
        return N;
    }
    int l = 0;
    int r = N;
    int result = 0;

    while(l <= r){
        int mid = l + (r - l) / 2;

        if(mid * mid <= N){
            return mid;
        }
        if(mid * mid < N){
            l = mid + 1;
            result = mid;
        }else {
            r = mid - 1;
        }
    }

    return result;
}

```
## Question:

There is an integer array nums sorted in ascending order (with distinct values).

Prior to being passed to your function, nums is possibly rotated at an unknown pivot index k (1 <= k < nums.length) such that the resulting array is [nums[k], nums[k+1], ..., nums[n-1], nums[0], nums[1], ..., nums[k-1]] (0-indexed). For example, [0,1,2,4,5,6,7] might be rotated at pivot index 3 and become [4,5,6,7,0,1,2].

Given the array nums after the possible rotation and an integer target, return the index of target if it is in nums, or -1 if it is not in nums.

```
Example 1:

Input: nums = [4,5,6,7,0,1,2], target = 0
Output: 4

Example 2:

Input: nums = [4,5,6,7,0,1,2], target = 3
Output: -1

Example 3:

Input: nums = [1], target = 0
Output: -1

```

## Brute Force Idea:

- **TC**: $O(N)$
- **SC**: O(1)

The simple brute force approach involves performing a linear search: iterate through the array, check each element, find the target, and return its index.

```java

public static int searchRotatedSorted(int[] nums, int target) {
    int N = nums.length;
    int ans = -1;

    for(int i = 0; i < N; i++){
        if(target == nums[i]){
            ans = i;
        }
    }

    return ans;
}

```

public int search(int[] nums, int target) {
        
    int N = nums.length;
    int l = 0, r = N - 1;

    while(l <= r){
        int mid = l + (r - l) / 2;
        if(nums[mid] == target){
            return mid;
        }
        if(nums[mid] >= nums[l]){
            if(nums[l] <= target && nums[mid] >= target){
                r = mid - 1;
            }else {
                l = mid + 1;
            }
        }else {
            if(nums[mid] <= target && nums[r] >= target){
                l = mid + 1;
            }else {
                r = mid - 1;
            }
        }
    }
    return -1;
}
