## Question :

Given a non-negative integer N, return the square root of N rounded down to the nearest integer. The returned integer should be non-negative as well.
Input: N = 4

```java

public static int sqrt(int N){

    // Edge case
    if (N == 0 || N == 1) {
        return N;
    }

    int i = 1;

    while(i * i <= N){
       i++;
    }

    return i - 1;

}

```

```java

public static int sqrt(int N){
     if (N == 0 || N == 1) {
        return N;
    }
    int l = 0;
    int r = N;
    int result = 0;

    while(l <= r){
        int mid = l + (r - l) / 2;

        if(mid * mid <= N){
            return mid;
        }
        if(mid * mid < N){
            l = mid + 1;
            result = mid;
        }else {
            r = mid - 1;
        }
    }

    return result;
}

```
